# Описание проекта
## В рамках задания реализованы два базовых класса для работы с каталогом товаров:
1. Product – представляет товар с его основными характеристиками.
2. Category – представляет категорию товаров, содержащую список продуктов.
### Реализация выполнена с учётом требований к атрибутам, инициализации и автоматическому подсчёту категорий и товаров.
### Класс Product описывает товар в магазине.
Атрибуты:
1. name — название товара
2. description — описание товара
3. __price — цена (приватный атрибут)
4. quantity — количество на складе
### Класс Category описывает категорию товаров.
Атрибуты:
1. name — название категории
2. description — описание категории
3. __products — список товаров (приватный)
