###Описание проекта###
##В рамках задания реализованы два базовых класса для работы с каталогом товаров:##
Product – представляет товар с его основными характеристиками.
Category – представляет категорию товаров, содержащую список продуктов.
Реализация выполнена с учётом требований к атрибутам, инициализации и автоматическому подсчёту категорий и товаров.
